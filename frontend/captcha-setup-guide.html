Step 1: Get reCAPTCHA Keys from Google

  1. 1 Go to Google reCAPTCHA Admin Console
  2. 2 Click "Create" to create a new site
  3. 3 Choose reCAPTCHA v2 ‚Üí "I'm not a robot" Checkbox
  4. 4 Add your domains (e.g., localhost, yourdomain.com)
  5. 5 Submit and copy your Site Key and Secret Key

Step 2: Configure Backend Environment

# Add to your .env file
RECAPTCHA_SECRET_KEY=your_secret_key_here

Replace your_secret_key_here with your actual secret key from Google.

Step 3: Add reCAPTCHA to Your Form

Option A: HTML Form (Simple)

<!-- Add reCAPTCHA script -->
<script src="https://www.google.com/recaptcha/api.js" async defer></script>
<!-- Add reCAPTCHA widget -->
<div class="g-recaptcha" data-sitekey="YOUR_SITE_KEY"></div>
<!-- Submit button -->
<button type="submit">Send Message</button>

Option B: JavaScript Form (Advanced)

// Get reCAPTCHA token before submission
const recaptchaToken = grecaptcha.getResponse();
// Add token to form data
formData.append('g-recaptcha-response', recaptchaToken);

Step 4: Enable CAPTCHA in Form Settings

  1. 1. Go to your form's Settings tab
  2. 2. Scroll to Security Settings
  3. 3. Toggle "Enable CAPTCHA" to ON
  4. 4. Click "Save" to apply changes

üîß Troubleshooting

Error: "reCAPTCHA token required"

Make sure you've added the reCAPTCHA widget to your form and completed the verification.

Error: "reCAPTCHA verification failed"

Check that your secret key is correctly set in the backend environment variables.

reCAPTCHA not showing

Verify your site key is correct and your domain is registered with Google reCAPTCHA.
